# Sample Obsidian Vault

This is a sample vault demonstrating the Obsidian markdown format used by LocalOS memory system.

## Structure

```
sample-vault/
├── Development/          # Development notes
├── Projects/            # Project documentation
├── Learning/            # Learning resources
├── Personal/            # Personal notes
└── Meetings/            # Meeting notes
```

## Features Demonstrated

- **Frontmatter**: YAML metadata at the top of files
- **Tags**: Both in frontmatter and inline with `#tag`
- **Links**: Wiki-style links with `[[Note Name]]`
- **Markdown**: Standard markdown formatting

## Usage

This vault can be:
1. Opened in Obsidian desktop app
2. Used for testing LocalOS memory features
3. Referenced as examples for creating new notes
