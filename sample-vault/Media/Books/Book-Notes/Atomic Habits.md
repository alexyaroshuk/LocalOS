# Atomic Habits - James Clear

## Summary
Small changes compound over time. The book argues that tiny improvements (1% better each day) lead to remarkable results. Focus on systems, not goals.

## Key Concepts

### The Habit Loop
1. **Cue** - Trigger that initiates behavior
2. **Craving** - Motivational force behind every habit
3. **Response** - Actual habit you perform
4. **Reward** - End goal of every habit

### The Four Laws of Behavior Change

**Make it Obvious** (Cue)
- Use implementation intentions: "I will [BEHAVIOR] at [TIME] in [LOCATION]"
- Stack habits: "After [CURRENT HABIT], I will [NEW HABIT]"
- Design your environment to make cues obvious

**Make it Attractive** (Craving)
- Pair actions you want to do with actions you need to do
- Join a culture where desired behavior is normal
- Create a motivation ritual

**Make it Easy** (Response)
- Reduce friction for good habits
- Prime your environment
- Use the Two-Minute Rule: scale habits down to 2 minutes
- Automate habits where possible

**Make it Satisfying** (Reward)
- Use immediate rewards
- Make "doing nothing" enjoyable (for breaking bad habits)
- Use a habit tracker
- Never miss twice

### Breaking Bad Habits
Invert the laws:
1. Make it Invisible
2. Make it Unattractive
3. Make it Difficult
4. Make it Unsatisfying

## My Takeaways & Applications

### Applied to Life

**Morning Routine** (Habit Stacking)
- Wake up → Make bed → Meditate → Workout → Shower → Coffee
- Each action triggers the next
- See: [[Personal/Health/Meditation Practice|Meditation Practice]]

**Fitness** (2-Minute Rule + Environment Design)
- Lay out gym clothes night before
- "Just put on gym clothes" → Usually leads to full workout
- Result: Went from 1x/week to 4x/week
- See: [[Personal/Health/Fitness Goals|Fitness Goals]]

**Coding/Learning** (Implementation Intentions)
- "After morning coffee, I will code for 30 minutes on LocalOS"
- Works remarkably well
- Built LocalOS v1.0 with this approach

**Reading** (Environment + Habit Stacking)
- Removed phone from bedroom
- Book on nightstand instead
- Read for 20 min before bed
- Result: 18 books read this year (was reading 4/year before)

**Reducing Phone Use** (Make it Difficult)
- Deleted social media apps
- Phone stays in other room while working
- Checking phone requires effort
- Screen time down 60%

### Identity-Based Habits

Instead of:
- ❌ "I want to read more" → ✅ "I am a reader"
- ❌ "I should work out" → ✅ "I am an athlete"
- ❌ "I need to code" → ✅ "I am a builder"

Every action is a vote for your identity.

## Related Concepts
- **Compound interest** - 1% better each day = 37x better in a year
- **Plateau of latent potential** - Valley of Disappointment
- **Sorites Paradox** - When does a pile of sand become a heap?

## Criticisms
- Some examples feel cherry-picked
- Not all habits compound equally
- Ignores role of willpower/discipline somewhat
- Can be overly prescriptive
