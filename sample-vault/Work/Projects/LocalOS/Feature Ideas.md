## Inspiration From
- **Letta** - Memory architecture (core + archival)
- **Obsidian** - Local-first, markdown notes
- **Perplexity** - Search integration
- **Rewind** - Comprehensive memory
- **NotebookLM** - Document analysis

## Technical Challenges
1. **Memory efficiency** - Running 8B models on phone
2. **Context window** - Managing limited context
3. **Search performance** - Fast vector search on device
4. **Battery life** - Inference is power-hungry
5. **Model updates** - Easy model switching
